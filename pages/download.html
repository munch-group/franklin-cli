Installation

Download and run the Installer. Before installing Franklin, the installer will download and install the following dependencies if they are not available available on your computer:

  1. A Python 3 distribution - preferably Miniforge.
  2. The Pixi, a modern, fast package manager thatâ€™s fully compatible with conda packages but offers better performance and reliability..
  3. The Chrome browser that Franklin uses to display JupyterLab and web pages.
  4. The Docker Desktop application that Franklin uses to containerize Jupyter environments.
